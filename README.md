# Soldadura_IA_XAI_Tesis
Tesis de grado:  Sistema Inspector de Defectos en Soldaduras. Incluye la documentación completa, archivos de código para el entrenamiento del modelo CNN 3D, y la inferencia del sistema. Además, se encuentra el Dataset completo de Tomografía Computarizada (TC) en formato DICOM y los metadatos de anotación en NIfTI
